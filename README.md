## Kratek opis projekta:
Spletna stran MicroHub je preprosta družbena platforma, zasnovana kot alternativa sodobnim družbenim omrežjem, ki so postala preobremenjena z nepotrebnimi funkcijami in oglaševanjem. Namen projekta je ponovno vzpostaviti jedro družbenih interakcij – deljenje objav, všečkanje, komentiranje ter prejemanje obvestil o dejavnostih drugih uporabnikov.

Problem, ki ga projekt MicroHub naslavlja, je prezasičenost sodobnih družbenih omrežij z nepotrebnimi funkcijami, oglaševanjem in algoritmi, ki zmanjšujejo pristnost interakcij med uporabniki. Zaradi tega se uporabniška izkušnja pogosto poslabša, družbena omrežja pa izgubljajo svoj prvotni namen – preprosto in neposredno povezovanje ljudi.
Rešitev, ki jo ponuja MicroHub, je minimalistična spletna platforma, ki se osredotoča na osnovne elemente družbene interakcije: objavljanje vsebin, všečkanje, komentiranje in obveščanje o dejavnostih drugih uporabnikov. S tem uporabnikom omogoča čistejšo, hitrejšo in bolj pristno komunikacijo brez motenj oglasov ali nepotrebnih funkcionalnosti.

## Ogrodje in razvojno okolje:

### Frontend:
React (TypeScript, Vite) – uporabniški vmesnik

### Backend (mikrostoritve):

Node.js (TypeScript) – izvajalno okolje za mikrostoritve

Express.js – REST API ogrodje za posamezne mikrostoritve

### Infrastruktura:

Docker – orkestracija okolja in izolacija storitev

### Razvojno okolje:

VS Code – razvoj

GitHub – repozitoriji in verzioniranje kode

## Seznam funkcionalnosti mikrostoritev:
### Users
Ustvarjanje, prijava in avtentikacija uporabnika

### Post
Kreairanje in upravljanje objav (CRUD opracije)

### Like
Mikrostoritev za všečkanje objav

### Comment
Mikrostoritev za komentiranje objav

### Notifications
Mikorstoritev za pošiljanje obvestil (npr. ko ti prijatelj všečka objavo, uporabnik dobi sporočilo)

### User Profile
Mikrostoritev za prikazovanje svojega osebnega profila oziroma profila prijateljev, ki vsebuje statistiko svojih/njegovih objav.

## Primeri uporabe:
Registracija, kreracja objave, lajk, komentar, ogled profila
